# Vital Theme

## About this theme
Organic, friendly, and approachable. Built for interfaces where warmth and accessibility are primary — health apps, wellness platforms, consumer products, community tools. Fresh without being clinical, human without being naive.

## Stack
- Next.js (App Router), TypeScript
- Tailwind CSS v4 + shadcn/ui
- Lucide React for icons
- Quicksand via next/font/google

## Design System
- Core rules: `../../core/PRINCIPLES.md` — read this first
- Full theme reference: `DESIGN.md` — read this for every UI task

## Rules
- Always follow PRINCIPLES.md and DESIGN.md before writing any component
- Use semantic CSS tokens only — never raw hex values
- Use shadcn components — never raw HTML equivalents
- Quicksand is the only typeface — never introduce a second font
- Use Lucide React only for icons
- Follow the spacing scale: 4 · 8 · 12 · 16 · 24 · 32 · 48px
- Favour rounded, soft corners — this theme is approachable not sharp
