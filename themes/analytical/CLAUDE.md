# Analytical Theme

## About this theme
Precise, warm, and data-first. Built for interfaces where information density and clarity are primary — dashboards, internal tools, ops platforms, data products. Warm enough to feel human, structured enough to handle complexity.

## Stack
- Next.js (App Router), TypeScript
- Tailwind CSS v4 + shadcn/ui
- Lucide React for icons
- DM Sans via next/font/google

## Design System
- Core rules: `../../core/PRINCIPLES.md` — read this first
- Full theme reference: `DESIGN.md` — read this for every UI task

## Rules
- Always follow PRINCIPLES.md and DESIGN.md before writing any component
- Use semantic CSS tokens only — never raw hex values
- Use shadcn components — never raw HTML equivalents
- DM Sans is the only typeface — never introduce a second font
- Use Lucide React only for icons
- Follow the spacing scale: 4 · 8 · 12 · 16 · 24 · 32 · 48px
- Favour density and precision over decoration — every element must earn its space
