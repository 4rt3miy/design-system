# Conversational Theme

## About this theme
Scholarly, warm, and reading-heavy. Built for interfaces where text is the primary medium — AI assistants, document tools, research platforms, knowledge bases. Draws from the tradition of structured beauty where clarity and elegance are inseparable.

## Stack
- Next.js (App Router), TypeScript
- Tailwind CSS v4 + shadcn/ui
- Lucide React for icons
- Crimson Pro via next/font/google

## Design System
- Core rules: `../../core/PRINCIPLES.md` — read this first
- Full theme reference: `DESIGN.md` — read this for every UI task

## Rules
- Always follow PRINCIPLES.md and DESIGN.md before writing any component
- Use semantic CSS tokens only — never raw hex values
- Use shadcn components — never raw HTML equivalents
- Crimson Pro is the only typeface — never introduce a second font
- Use Lucide React only for icons
- Follow the spacing scale: 4 · 8 · 12 · 16 · 24 · 32 · 48px
- When in doubt, favour whitespace over density
